# vscode-office

This extension is to support the function of previewing common office file formats.

It support below files now:
- Excel: .xls, .xlsx
- Word: .docx
- PhotoShop: .psd
- Svg: .svg
- Pdf: .pdf
- Epub: .epub
- Xmind: .xmind
- PlantUml: .puml, .pu, .plantuml
- Windows Reg: .reg
